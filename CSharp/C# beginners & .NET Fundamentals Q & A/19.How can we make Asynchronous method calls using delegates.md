# Asynchronous Method Calls Using Delegates (Classic .NET Framework)

## What is Asynchronous Calling?

In a synchronous call, the caller waits until the method completes.

In an asynchronous call, the caller does not wait. Control returns immediately to the caller while the task continues in the background and later triggers a callback.

## Key Idea

1. Control returns immediately to the client.
2. The client continues its work.
3. The long-running task executes independently.
4. When the task completes, a callback method is triggered.
5. The callback retrieves the result.

---

## Step 1: Create a Delegate

```csharp
public delegate string PointerToHugeTask();
```

---

## Step 2: Create the Long Running Task

```csharp
class CLHugeTask
{
    public string DoHugeTask()
    {
        Thread.Sleep(5000);

        return "Task completed at " +
               DateTime.Now.ToLongTimeString();
    }
}
```

---

## Step 3: Callback Method

```csharp
public static void CallbackMethod(IAsyncResult result)
{
    PointerToHugeTask ptr =
        (PointerToHugeTask)((AsyncResult)result).AsyncDelegate;

    string value = ptr.EndInvoke(result);

    Console.WriteLine(value);
}
```

### What Happens Here?

1. .NET calls CallbackMethod.
2. Retrieve the delegate from IAsyncResult.
3. Call EndInvoke().
4. Get the return value from DoHugeTask().

---

## Step 4: Invoke Asynchronously

```csharp
static void Main()
{
    CLHugeTask obj = new CLHugeTask();

    PointerToHugeTask ptr =
        new PointerToHugeTask(obj.DoHugeTask);

    ptr.BeginInvoke(
        CallbackMethod,
        null);

    Console.WriteLine("Waiting in Main Method");

    Console.ReadLine();
}
```

---

## Important Methods

### BeginInvoke()

```csharp
ptr.BeginInvoke(callback, state);
```

- Starts execution asynchronously.
- Does not block the caller.
- Returns immediately.

### EndInvoke()

```csharp
string result = ptr.EndInvoke(asyncResult);
```

- Retrieves the return value.
- Waits for completion if necessary.
- Cleans up resources.

## Complete Program

```csharp
using System;
using System.Runtime.Remoting.Messaging;
using System.Threading;

public delegate string PointerToHugeTask();

class CLHugeTask
{
    public string DoHugeTask()
    {
        Thread.Sleep(5000);

        return "Task completed at " +
               DateTime.Now.ToLongTimeString();
    }
}

class Program
{
    public static void CallbackMethod(IAsyncResult result)
    {
        PointerToHugeTask ptr =
            (PointerToHugeTask)
            ((AsyncResult)result).AsyncDelegate;

        string value = ptr.EndInvoke(result);

        Console.WriteLine(value);
    }

    static void Main()
    {
        CLHugeTask obj = new CLHugeTask();

        PointerToHugeTask ptr =
            new PointerToHugeTask(obj.DoHugeTask);

        ptr.BeginInvoke(
            CallbackMethod,
            null);

        Console.WriteLine("Waiting in Main Method");

        Console.ReadLine();
    }
}
```

## Interview Questions

### What is asynchronous method invocation using delegates?

A delegate executes a method on a separate thread using BeginInvoke(), allowing the caller to continue execution without waiting.

### What is the difference between Invoke and BeginInvoke?

| Invoke | BeginInvoke |
|----------|------------|
| Synchronous | Asynchronous |
| Caller waits | Caller continues immediately |
| Blocks thread | Non-blocking |
| Returns result directly | Result obtained using EndInvoke |

### Why is EndInvoke required?

- Retrieves the return value.
- Waits for completion if required.
- Cleans up resources.

### Is BeginInvoke still recommended?

No. It is a legacy .NET Framework asynchronous pattern.

Modern .NET uses:

- Task
- async
- await
- Task.Run()

Example:

```csharp
public async Task<string> DoHugeTask()
{
    await Task.Delay(5000);
    return "Task Completed";
}
```

## Interview Summary

### Legacy Pattern

```text
Delegate
   |
   +--> BeginInvoke()
           |
           +--> Method executes asynchronously
           |
           +--> Callback Method
                   |
                   +--> EndInvoke()
                   |
                   +--> Get Result
```

### Modern Pattern

```text
Task
  |
  +--> async
          |
          +--> await
                  |
                  +--> Continuation
```
